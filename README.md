# Iris-species-model
Iris Flower Classification 

This is a simple Machine Learning project that predicts the species of an iris flower using the Iris dataset.

 PROJECT OVERVIEW.

The goal of this project is to build a machine learning model that can classify iris flowers into three species based on their features.

The three iris species are:

- Setosa
- Versicolor
- Virginica

The model uses flower measurements to predict the correct species.

 Dataset

The dataset used in this project contains the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The target column:

- Species

Total records: 150

 Libraries Used.

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

 Machine Learning Model

The model used in this project is:

- Logistic Regression -> is used when the output is categorical.

Steps performed✔ :-

1. Import libraries
2. Load dataset
3. Encode target labels
4. Split data into training and testing sets
5. Train the model
6. Predict results
7. Evaluate accuracy

 Model Accuracy

The model achieved an accuracy of:

1.0 (100%)
